About This Project

The Cat vs Dog Classifier is a machine learning-based web application designed to predict whether an uploaded image contains a cat or a dog.

By analyzing key features in the image, the model helps users quickly identify the type of pet in the picture.

Built using Flask, Python, and Tailwind CSS, the app provides an intuitive user experience and showcases how artificial intelligence can simplify and enhance image classification tasks.
